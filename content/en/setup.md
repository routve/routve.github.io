## Installation

Add `routve` dependency to your project:

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn add routve
  ```

  </code-block>
  <code-block label="NPM">

  ```bash
  npm install routve
  ```

  </code-block>
</code-group>
